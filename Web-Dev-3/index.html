1. SEO (Search Engine Optimization): The process of improving a website’s visibility in search engine results to increase organic traffic by optimizing content, keywords, and user experience.

2. Cumulative Layout Shift (CLS): A metric that measures unexpected layout shifts on a webpage while loading. A low CLS means a stable layout, enhancing user experience.

3. Largest Contentful Paint (LCP): A metric that tracks how quickly the largest visible content element (e.g., image or text) loads. A faster LCP means the page becomes usable sooner.

4. First Input Delay (FID): A metric that measures the time it takes for a webpage to respond to the first user interaction (e.g., clicking a link or button). A lower FID indicates a more responsive and user-friendly page. It helps evaluate how quickly users can start interacting with a website.
