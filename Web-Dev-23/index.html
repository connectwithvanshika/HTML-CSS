CSS Position Property

The CSS positions allow you to precisely control the placement of an element on the web page.

  • Static Position

    • *The elements are positioned according to the normal flow of the document.

  • Relative position

    • *Elements are positioned relative to the normal position in the document. You can use the top, right, bottom, and left properties to move the element from its original position.

    Z-index

    z-index is a CSS property used to control the vertical stacking order of elements that overlap one another. It only works on elements that have a position value of relative, absolute, fixed, or sticky. By default, elements are stacked in the order they appear in the HTML code, with later elements appearing on top of earlier ones.
